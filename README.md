# <p align="center">[uniapp template](https://github.com/zhanbai/uniapp-template)</p>

## 关于

uniapp 是一个使用 Vue.js 开发所有前端应用的框架，开发者编写一套代码，可发布到多个平台。本项目旨在依托 uniapp 搭建最基础的、最通用的功能，方便快速开发一个新的项目。

## 使用
```
npm install
```

### 开发环境编译和热重载
```
npm run serve
```

### 生产环境编译和压缩
```
npm run build
```
